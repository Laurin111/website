Ich habe mithilfe von Java Swing einen Taschenrechner programmiert, der mit Tastatureingabe und mit Knöpfen funktioniert. Die Validierungen dafür haben mit abstand am längsten gedauert. Jeder Fall muss behandelt werden. An diesem Beispiel sieht man die Schwierigkeiten des Programmierens sehr gut.

Ein weiteres meiner Projekte ist das TicTacToe. Dort musste ich einen Computergegner programmieren, der jedes Mal, wenn ich einen Spielzug gemacht habe, mit einem anderen Spielzug kontert. Ebenfalls kann man so lange spielen, bis man keine Lust mehr hat und man kann den Computer auch den ersten Spielzug machen lassen.

Wie bei der Adressverwaltung erwähnt, habe ich auch schon vieles mit SQL gemacht. Sei es eine Java Schnittstelle, sei es eine JPA-Anbindung oder wie gesagt eine Adressverwaltung. Im betrieb lernen wir viele verschiedene Varianten davon kennen.

Einer meiner kürzlichen Projekte eine Adressverwaltung mit Anbindung an eine Datenbank. Bei der Verwaltung kann man Personen mit bestimmten Adressen erstellen, bearbeiten und löschen. Dafür habe ich eine Datenbank erstellt und zwei unterschiedliche arten von Schnittstellen programmiert.
